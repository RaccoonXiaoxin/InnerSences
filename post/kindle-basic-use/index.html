Kindle 的基本使用

· 2019-03-04 · # Kindle

  作为 Amazon 出品的电纸书，Kindle 系列在取代部分纸质书的道路上还是取得了不小的进步。笔者从 2012 年入手 KPW2 后，基本上大部分的课外书都是在上面读完的。作为一名老用户，在此介绍一些 Kindle 的日常使用。

Kindle 的系统是基于 Linux 内核做的二次开发，整体操作风格和手机的系统差别并不大，系统本身支持一部分命令的输入。

  • 同时轻触左上角和右下角（或者右上角和左下角）的屏幕可以截屏，用数据线连接电脑后查看根目录，就能得到已保存的截屏图片。
  • 把文字版 PDF 采用邮件方式发送到 Kindle 的时候，邮件主题写 convert，系统会自动把 PDF 转换成适合 Kindle 阅读的格式。扫描版的 PDF 不行。
  • 选中一段文字后点击 "标注"，可以保存这段文字，查看时点击书架上的“您的剪藏”即可。
  • 点击主页上的菜单 - 新建收藏夹，即可为图书进行个性化分类。
  • 只要买过亚马逊 Matchbook 项目里的实体书，就可以免费或低价购买该实体书的相应电子书。
  • 随身携带 Kindle 的话，手机没有电了可以找一个有 Wifi 的地方连上网页版 QQ 找人。
  • 打开 WLAN 联网后，选中词汇可以查看相应的维基百科解释。
  • 可以在设置 - 设备选项 - 语言和字典 - 字典中修改默认的字典设置。
  • 在主页点击菜单 - 设置 - 阅读选项 - 语言习得 - 生词提示，可选择开启或关闭生词提示的选项。
  • 手指在屏幕上捏合或张开可以缩放字号。
  • 拥有 X-Ray 功能的 Kindle ，可以查看选中人名或短语在全书中出现过的位置列表，相当于索引。很多纸书编辑偷懒没有做的功能，Kindle 总算轻松实现。
  • 关闭 Kindle 锁屏屏保：点击搜索框，输入~ds，然后锁屏屏保就关闭了。关闭后，无论闲置多长时间还是点按电源键，都无法关闭屏幕，但使用磁铁关闭屏幕依然可以使用。重启后失效。
  • 阅读图片：在根目录下建立一个 images 文件夹，然后再在下面建立一个文件夹，命名随意，作为一个分组，然后将图片放到这个文件夹下即可。最佳尺寸：宽 1563 像素，长 2500 像素。也可以打包成 zip 文件，通过邮件直接推送到设备上。
  • 安装第三方字典：将 Kindle 连接计算机，将下载的第三方字典 mobi 文件放到 documents\dictionaries\ 目录下，然后断开连接，在字典的设置中勾选相应的字典使用。
  • 忘记锁屏密码：没有任何方式可以实现重置密码，只能选择重置设备。在密码输入框中输入 111222777（三个1，三个2，三个7），点击“确定”后设备就会自动开始重置并重启
