Windows 注册表的日常使用

· 2019-03-11 · # Windows

  日常使用 Windows 系统，注册表是个无法跳过的内容，本文将介绍下注册表的基本使用，着重讲解下命令行模式下如何进行注册表添加、删除、查看等操作。

零、引言

  Windows 系统的注册表相当于系统级配置信息数据库 ，大部分系统设置和软件配置信息都放在注册表里面，而注册表的破坏严重时可能直接导致 Windows 系统的崩溃，因此，在着手修改注册表之前记得做好备份，方便操作错误时恢复注册表。
  注册表编辑器 reg.exe，一般位于系统盘的 C:\Windows\System32\reg.exe 。

一、图形化注册表编辑器

  图形化注册表编辑器的打开方式有三种：

  1. 开始菜单，直接搜索“regedit”或“注册表编辑器”
  2. 运行 界面，输入“regedit”
  3. 命令行 界面，输入“regedit”

  图形化注册表编辑器的界面如下图，键、值和类型三个名词就是分别表示对应位置的内容。

备份操作在上方的菜单，文件--导出

二、命令行注册表编辑器

  实际上图形界面的注册表编辑器可以完成所有注册表编辑操作，在日常的 Windows 使用过程中都可以满足修改注册表的需要。但有时并没有图形界面，比如只有一个 shell，这时想修改注册表的值，就必须使用命令行注册表编辑器了。
要调用 "控制台注册表编辑器" 只需要打开 CMD 命令行，输入 "reg /?" 就能看到帮助文件了。

命令行本身不区分大小写。

三、常用的使用和相关操作

  1. 调整 Windows 10「菜单」视觉效果显示速度延迟
    Windows 10 系统中有一系列的动画视觉效果，能够让用户的 UI 体验更加流畅，其中一项就包括打开菜单时的平滑过渡延迟效果。对绝大多数用户来说，系统默认设置都是比较理想的，不过您还是可以进一步加速菜单视觉效果延迟，让其感觉起来更加快速。

    HKEY_CURRENT_USER\Control Panel\Desktop

    修改右侧 MenuShowDelay 键值，默认是 400 毫秒。
    若要加快动画速度，使用小于 400 的数值，如 75、100 或 125 等，值设置为 0 时，将完全禁用菜单视觉效果。
    若要减慢动画效果速度，请使用大于 400 的值，但最大不超过 4000。
    值设置好后重启计算机即可查看到效果。

  2. 关闭 Windows 10 两分钟后自动进入睡眠
    Windows 10 系统为了能够让轻薄本进一步增强续航，增加了一个隐藏的电源管理项，2 分钟如果没有操作系统将会自动进入休眠状态。这对于台式机或者连着电源使用的笔记本并没有太大的意义，可以通过如下的方式进行修改。

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\238C9FA8-0AAD-41ED-83F4-97BE242C8F20\7bc4a2f9-d8fc-4469-b07b-33eb785aaca0

    修改右侧 Attributes 键值，将数值数据由默认值 1 修改为 2 ，确定。
    电源选项中的 “更改高级电源设置” 即可打开 “电源选项 - 高级设置” 窗口。点击 + 号展开 “睡眠”，即可看到“无人参与系统睡眠超时” 设置项，默认 2 分钟修改为一个较大的数值，起码不小于你在 “Windows 设置 － 系统 － 电源和睡眠” 设置的睡眠超时时间即可。

  3. Windows 10 内测版本改为“跳到下一个 Windows 版本”
    选择参加 Windows 预览体验计划的用户，会发现“跳过”选项无法勾选，那是因为微软本身针对跳到最新大版本测试下有着严格的筛选，不满足要求的用户就无法勾选，但是可以通过修改注册表来实现。

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost\Applicability

    右侧 ContentType 的键值从 Active 改成 Skip
    更改后再检查更新就会收到下一个版本的更新了，更新安装好后重启完成，也可以看到“跳过”选项可以正常使用了。

  4. 远程桌面记录的管理

    HKEY_CURRENT_USER\Software\Microsoft\Terminal Server Client\Default

    右侧的值都是连接过的远程桌面 IP 地址，可以自行将不需要的删除掉，这样远程桌面的下拉菜单中就没有对应的记录了

  5. 恢复照片查看器打开图片文件
    笔者一直很喜欢 Windows 7 时代的照片查看器，对于 Windows 10 上自带的照片应用始终用着不顺手，可以直接命令行执行下述的内容

    reg add "HKLM\SOFTWARE\Microsoft\Windows Photo Viewer\Capabilities\FileAssociations" /v ".jpg" /t REG_SZ /d PhotoViewer.FileAssoc.Tiff /f
reg add "HKLM\SOFTWARE\Microsoft\Windows Photo Viewer\Capabilities\FileAssociations" /v ".jpeg" /t REG_SZ /d PhotoViewer.FileAssoc.Tiff /f
reg add "HKLM\SOFTWARE\Microsoft\Windows Photo Viewer\Capabilities\FileAssociations" /v ".bmp" /t REG_SZ /d PhotoViewer.FileAssoc.Tiff /f
reg add "HKLM\SOFTWARE\Microsoft\Windows Photo Viewer\Capabilities\FileAssociations" /v ".png" /t REG_SZ /d PhotoViewer.FileAssoc.Tiff /f
reg add "HKLM\SOFTWARE\Microsoft\Windows Photo Viewer\Capabilities\FileAssociations" /v ".cr2" /t REG_SZ /d PhotoViewer.FileAssoc.Tiff /f
reg add "HKLM\SOFTWARE\Microsoft\Windows Photo Viewer\Capabilities\FileAssociations" /v ".jpg_large" /t REG_SZ /d PhotoViewer.FileAssoc.Tiff /f

    其他图片类型可以类似增加

  6. 恢复音量合成器旧版样式
    Windows 10 新版音量合成器修改起来有些困难，想要恢复旧版的样式可以如下修改

    reg add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\MTCUVC"
reg add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\MTCUVC" /v EnableMTCUVC /t REG_DWORD /d 0

  8. 关闭 Windows 10 默认 C 盘保留 7 GB 空间用于更新
    Windows 10 增加了一个磁盘空间的控制，C 盘会在默认情况下保留了 7 GB 的空间用于系统升级，这本身是个好事，避免因为 C 盘空间不足导致整个系统在升级过程中卡死而只能格式化重装。但是对于小硬盘的用户来说，本来就只有一点点大小空间的 C 盘，7 GB 还不允许使用，简直就是灾难了。

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\ReserveManager

    右侧 ShippedWithReserves 的键值从 0 改成 1
