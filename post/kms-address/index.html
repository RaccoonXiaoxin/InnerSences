目前可用的 KMS 地址
KMS，Key Management Service，Windows 密钥管理服务，在 Windows Vista 之后的产品中的一种新型产品激活机制，目的是为了 Microsoft 更好的遏制非法软件授权行为。下面罗列了目前可用的 KMS 地址，便于日常激活 Windows 系统和 Office 套件时使用。
